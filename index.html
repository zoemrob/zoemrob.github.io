A little about me...

Self-taught. Ambitious. Developed in Portland, OR.

I love to code. Whether it's the latest article on Medium, planning api endpoints on my phone, writing scripts to automate an existing process, or thinking about the next middleware opportunity, you can bet there's some new programming idea on my mind.

I had always been interested in programming, but without college, I thought I didn't have a shot. Browsing the internet one day, I recall reading an article containing a statistic about how by 2020 there will be a projected 1,000,000 more computing related jobs than students with Computer Science degrees. "Wow!" I thought, "Maybe I could do that!" The myriad of free, or low-cost, learning resources was overwhelming at first, but I landed on Flatiron School Bootcamp Prep as my first structured program.

Initially it was so challenging! Getting into the "programming mindset" I think is the hardest initial challenge. My - "I think I can actually do this" - moment was when I was handed an assignment at work to obtain a list of codes (essentially key, value pairs) from a public website. Turns out there were 800+ webpages containing the data! I saw this as an opportunity to use code to solve an otherwise ridiculously time-consuming process. A few hours later, I was able to run my script and output a json formatted snippet of all of the codes, obtained from dynamically-generated urls and extracted using regular expressions from the document. It was ugly. It definitely didn't follow good coding practices, but it worked.
